可讓wooCommerce, Easy Digital Downloads 以及myCred使用綠界、藍新、LinePay等付款方式.

== Description ==

本外掛可讓綠界、藍新、Line Pay都可使用以上系統. 未來還可以支持更多的電商系統以及付費方式.
本外掛著重於虛擬商品銷售如線上教學、線上服務等.

== Installation ==

安裝程序:

1. 停用本外掛如果您安裝了前一版本.
2. 解壓縮下載的ZIP壓縮檔案內容到目錄\"/wp-content/plugins/zj-payment-packs"之下.
3. 透過WordPress管理頁面中的'外掛'選單啟用\"ZJ金流\".
4. 到[設定]-[ZJ金流]選單項目然後選取訂單系統以及付費方式.

== Frequently Asked Questions ==

- 在我安裝完外掛以後接下來呢?
到[設定][ZJ金流]然後啟用您的wordpress訂單系統如wooCommerce, Easy Digital Downloads, buycred以及支付方式.

- 為何我不能啟用綠界或者藍新的付費方式?
啟用付費方式, 有些欄位必須填寫完成才能啟用.

- 付費中的測試模式是什麼?
測試模式是模擬真實交易情境下之的之交易方式, 建議在進行真實交易之前先測時模式, 如果沒有問題在進入真實交易模式.
